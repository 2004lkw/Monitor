About Monitor

  • Green LED = All records indicate this host is reachable and stable.
  • Yellow LED = There has been at least one failed attempt to reach this host.
  • Red LED = This host is unreachable at the most recent attempt.

Each block is a representation of a unit of time. Red blocks indicate a failed attempt. Green blocks represent success. Values that appear when you hover over a block indicate the estimated time that attempt was made.

You can add and remove hosts by clicking on Hosts Edit. (adding and removing hosts can take a few minutes to take effect.) In addtion, clicking Refresh Now can speed the process up a bit most times. Using the Monitory Summary, you can filter the table by color by clicking the number of the status or clicking the total.

To fine tune other settings such as duration, time between checks and others, please view the readme that was distributed with this package.
